# GenAIGD_GarifullinAidar

## Жанр

Экшен‑рогалик с быстрыми забегами, боевыми комнатами и головоломками, основанными на переключении между обычным и зеркальным мирами.

## Уникальность проекта (USP)

Каждый этаж существует в двух версиях, и игрок свободно переключается между ними, чтобы сражаться, обходить ловушки и решать пространственные задачи. Зеркальный мир опаснее, но открывает скрытые маршруты, редкие ресурсы и уникальные механики, которые меняют стиль прохождения.

## Целевая аудитория

Игроки, которым нравятся динамичные рогалики, вариативность забегов, риск‑менеджмент, а также головоломки, встроенные в боевой процесс. Подходит для фанатов быстрых экшенов и игр с параллельными мирами.

## Платформы

PC (Windows/Linux)
